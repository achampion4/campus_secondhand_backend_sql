## 七、订单模块（OrderController）

|---|---|---|---|---|

| O-03 | 🔒 /api/order | POST | 购买已售/已下架商品 | code=500，提示"商品已售出或已下架" |
| O-04 | 🔒 /api/order | POST | 不存在的 productId | code=500，提示"商品不存在" |
| O-05 | 🔒 /api/order/bought | GET | 携带 token | code=200，返回我买到的订单列表(含明细) |
| O-06 | 🔒 /api/order/sold | GET | 携带 token | code=200，返回我卖出的订单列表(含明细) |
| O-07 | /api/order/detail/{id} | GET | 存在的订单ID | code=200，返回订单详情(含明细+商品标题) |
