## 一、用户模块（UserController）

| 编号 | 接口 | 方法 | 输入 | 预期结果 |
|---|---|---|---|---|
| U-01 | /api/user/register | POST | 合法用户名/密码/昵称 | code=200，返回新用户ID |
| U-02 | /api/user/register | POST | 已存在的用户名 | code=500，提示"用户名已存在" |
| U-03 | /api/user/register | POST | 密码少于6位 | code=400，参数校验失败 |
| U-04 | /api/user/register | POST | 用户名为空 | code=400，参数校验失败 |
| U-05 | /api/user/login | POST | 正确用户名+密码 | code=200，返回 token 与 user |
| U-06 | /api/user/login | POST | 错误密码 | code=500，提示"密码错误" |
| U-07 | /api/user/login | POST | 不存在的用户名 | code=500，提示"用户不存在" |
| U-08 | /api/user/login | POST | 被封禁账号（status=0） | code=500，提示"账号已被封禁" |
| U-09 | 🔒 /api/user/profile | GET | 携带有效 token | code=200，返回个人信息(无密码) |
| U-10 | 🔒 /api/user/profile | GET | 不带/错误 token | code=401，未登录 |
| U-11 | 🔒 /api/user/profile | PUT | 修改昵称/头像 | code=200，再查询昵称已更新 |

## 七、后台管理模块（AdminController）

> 以下接口均需管理员 token，role=1。使用管理员账号（admin / 123456）登录获取 token。

### 7.1 用户管理

| 编号 | 接口 | 方法 | 输入 | 预期结果 |
|---|---|---|---|---|
| AM-01 | 🔑 /api/admin/users | GET | 管理员 token | code=200，返回全部用户列表(无密码) |
| AM-02 | 🔑 /api/admin/users | GET | 普通用户 token | code=403，无权限 |
| AM-03 | 🔑 /api/admin/user/{id}/status | PUT | status=0 封禁用户 | code=200，该用户无法登录 |
| AM-04 | 🔑 /api/admin/user/{id}/status | PUT | status=1 解封用户 | code=200，该用户可正常登录 |

### 7.2 商品管理

| 编号 | 接口 | 方法 | 输入 | 预期结果 |
|---|---|---|---|---|
| AM-05 | 🔑 /api/admin/products | GET | 管理员 token | code=200，返回全部商品(含下架/已售) |
| AM-06 | 🔑 /api/admin/product/offShelf/{id} | PUT | 强制下架任意商品 | code=200，商品状态变为 0 |

### 7.3 举报处理

| 编号 | 接口 | 方法 | 输入 | 预期结果 |
|---|---|---|---|---|
| AM-07 | 🔑 /api/admin/reports | GET | 管理员 token | code=200，返回全部举报(带商品标题、举报人) |
| AM-08 | 🔑 /api/admin/report/{id} | PUT | status=1 处理举报 | code=200，举报状态变为"已处理" |
| AM-09 | 🔑 /api/admin/report/{id} | PUT | status=2 驳回举报 | code=200，举报状态变为"已驳回" |
| AM-10 | 🔑 /api/admin/report/{id} | PUT | status=3（非法值） | code=500，提示"处理状态非法" |

### 7.4 数据统计

| 编号 | 接口 | 方法 | 输入 | 预期结果 |
|---|---|---|---|---|
| AM-11 | 🔑 /api/admin/stats | GET | 管理员 token | code=200，返回统计 JSON |
| AM-12 | 🔑 /api/admin/stats | GET | — | 返回字段含 userCount, productCount, orderCount, totalSales, hotCategories, activeUsers |
