## 10. 业务流程端到端测试

### 流程一：交易流程

```
注册 → 登录 → 发布商品 → 买家浏览 → 买家下单 → 卖家发货 → 买家收货 → 双方互评
```

### 流程二：社交流程

```
买家收藏商品 → 买家联系卖家（聊天消息）→ 收藏列表查看
```

### 流程三：治理流程

```
用户举报违规商品 → 管理员查看举报列表 → 管理员确认违规（自动下架商品）→ 管理员封禁恶意用户
```

### 流程四：AI 导购流程

```
游客输入需求 → DeepSeek 分析商品库 → 返回推荐商品+理由 → 游客点击查看详情
```
